## Features

- Real-time messaging
- One on one and group chat conversations
- Android-only deployment
- Implementation using TypeScript
- Consistent coding standards using ESLint and Prettier
- Unit testing with Jest

## Getting Started

## Prerequisites

- Node.js (v16 or later)
- npm or yarn
- Expo CLI
- Android Studio and Android Emulator
npm install -g expo-cli

### Installation
git clone https://github.com/ayberk-kara/HowudoinApp.git
cd HowudoinApp
npm install


### Running on Android
npm start

### Testing

npm test
